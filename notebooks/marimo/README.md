# Marimo Notebooks

Interactive [Marimo](https://marimo.io/) notebooks. Run locally or open in [Molab](https://molab.marimo.io/) (free cloud).

## Core Notebooks

| Notebook | What it covers |
|----------|---------------|
| **01_company.py** | Company -- sections, show, trace, diff, BS/IS/CF, ratios, EDGAR |
| **02_scan.py** | Scan -- screen, scanAccount, scanRatio, benchmark, insights, rank |
| **03_ask.py** | Ask -- AI analysis, streaming, chat, setup |

## Run locally

```bash
uv add dartlab marimo
marimo edit notebooks/marimo/01_company.py
marimo edit notebooks/marimo/02_scan.py
marimo edit notebooks/marimo/03_ask.py
```
