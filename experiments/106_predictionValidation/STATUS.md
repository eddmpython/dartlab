
## 시장 엔트로피 실험 (004)

### 관측된 시장 엔트로피

| 기간 | H_norm | up비율 | 중앙값 |
|------|--------|--------|--------|
| 2021→2022 | 0.797 | 46.5% | -1.5% |
| 2022→2023 | 0.782 | 53.8% | +2.7% |
| 2023→2024 | 0.743 | 53.1% | +2.6% |

**엔트로피가 0.74~0.80으로 일관되게 높음** → 시장은 항상 분산 상태. 집중 상태(H < 0.6)가 관측되지 않아 엔트로피 기반 분기가 작동하지 않음.

### 결과 (936 관측치)

| 전략 | 정확도 |
|------|--------|
| OLS | 48.5% |
| **"항상 up"** | **53.1%** |
| 엔트로피 적응 | 48.5% (= OLS와 동일, H_norm > 0.75이므로) |
| 시장+기업 결합 | 53.1% (= "항상 up"과 동일) |

## 최종 결론

### 1. 연간 재무 데이터로 방향 예측은 본질적으로 어렵다

- **모든 전략이 50±3% 범위** — 동전 던지기와 통계적으로 구분 불가
- 가장 "좋은" 전략은 **"항상 상승 예측"(53.1%)** — 단순히 시장의 상승 편향을 반영
- 이것은 전략이 아니라 **시장 base rate**

### 2. 왜 예측이 안 되는가

| 원인 | 설명 |
|------|------|
| **연간 데이터 6개** | 기업당 데이터 포인트가 극소. OLS에 3-4개 점을 넣으면 R²가 무의미 |
| **비정상성** | COVID(2020), 반도체 사이클 등 구조적 변화가 매년 발생 |
| **전이확률 = base rate** | 전년 방향이 다음 해 방향의 정보를 거의 담지 않음 (conditional ≈ unconditional) |
| **엔트로피 일정** | 시장은 항상 높은 분산 상태 → 엔트로피가 변별력 없음 |

### 3. dartlab에 대한 시사점

> **방향 예측 자체에 수학적 방법론을 추가해도 개선이 없다.**
> 기존 파이프라인의 예측력이 낮은 것은 방법론 문제가 아니라 **데이터 한계**.

실질적 개선 경로:
1. **분기별 예측으로 전환** — 데이터 포인트 4배 (20+개). 칼만 필터 등 시계열 기법의 최소 요건 충족
2. **방향 대신 확률/범위 제시** — "상승 53%" 같은 확률 표현이 "상승 예측"보다 정직
3. **"예측하지 않겠다"도 답** — 데이터 한계 내에서 정직한 불확실성 표현이 거짓 확신보다 낫다

### 4. 조사했던 고급 방법론에 대한 최종 판정

| 방법론 | 적용 가능성 | 이유 |
|--------|-----------|------|
| Hurst 지수 | ❌ | 20+ 데이터 필요, 연간 6개로 불가 |
| 칼만 필터 | ⚠️ | 분기별 전환 시 가능 (20+ 분기 데이터) |
| Shannon 엔트로피 | ❌ | 시장 엔트로피가 일정하여 변별력 없음 |
| 전이확률 | ❌ | unconditional = conditional, 정보량 0 |
| TDA / 웨이블릿 | ❌ | 100+ 데이터 필요, 근본적으로 불가 |
| 앙상블 가중 최적화 | ⚠️ | 검증 구간 부족 (2개). 분기별 전환 시 재검증 가치 |

## 문서 텍스트 신호 실험 (005)

**데이터**: 100개 기업, 2023 문서 신호 → 2024 매출 변화

### 현재 인프라의 한계: 변화율이 변별력 없음

| 관측 | 의미 |
|------|------|
| 98/100 기업이 "negative" 신호 | 모든 기업의 리스크 섹션이 매년 크게 바뀜 → **신호가 아니라 잡음** |
| 전체 변화율이 대부분 10-30% 구간 | 분포가 좁아서 구간별 차이 관찰 불가 |
| 사업내용/매출관련 100% "매우높음(60%+)" | **모든 기업이 동일 구간** → 변별력 제로 |

### 핵심 질문: 문서 변화 = 매출 방향 전환?

| 그룹 | 전환율 | n |
|------|--------|---|
| 전체 | 48.4% | 93 |
| 변화율 상위 25% | 52.2% | 23 |
| 변화율 하위 25% | 43.5% | 23 |
| **차이** | **+8.7%p** | — |

**+8.7%p 차이**: 방향은 맞지만 표본 23개로는 통계적 유의성 미달. 더 큰 표본이 필요.

### 왜 현재 방식이 작동하지 않는가

1. **해시 기반 변화 감지**: 한 글자만 바뀌어도 "변화"로 기록 → 거의 모든 토픽이 매년 변화
2. **크기만 측정, 방향 없음**: "위험이 늘었는지 줄었는지"가 아니라 "뭔가 바뀌었는지"만 측정
3. **모든 기업이 같은 구간**: 공시 형식이 유사하여 변화율 분포가 좁음

### 다음 단계 (이 실험의 진짜 가치)

현재 `calcDisclosureDelta`의 "변화율"은 방향 정보가 없어서 예측에 쓸 수 없다.
학술 연구가 검증한 실제 작동하는 신호는 **변화의 방향**이다:

| 신호 유형 | 측정 방법 | dartlab 인프라 | 상태 |
|-----------|-----------|---------------|------|
| **위험 항목 추가/삭제** | diff 줄 단위에서 +/- 카운트 | `c.diff(topic, from, to)` 존재 | 🔧 추출 로직 필요 |
| **사업 확장/축소 언어** | 신규 키워드 출현 ("착공", "진출") | `keywordTrend()` 존재 | 🔧 키워드 설계 필요 |
| **텍스트 길이 변화 방향** | 섹션 길이 증가 vs 감소 | diff history에 `fromLen, toLen` 존재 | 🔧 집계 로직 필요 |
| **톤 분석** | 긍정/부정 단어 비율 | ❌ 미구현 | 📐 한국어 금융 사전 필요 |

**결론**: "문서가 바뀌었는가"가 아니라 **"문서가 어떤 방향으로 바뀌었는가"**를 추출해야 한다.
이것이 106 실험의 핵심 발견이다.

## disclosureRisk 시그널 실험 (009)

**데이터**: changes.parquet 전 종목, 2개 연도 교차 검증

### disclosureRisk 종합 등급 — 유일하게 두 연도 일관

| 등급 | →2023 (n) | →2023 down | →2023 평균 | →2024 (n) | →2024 down | →2024 평균 |
|------|-----------|-----------|-----------|-----------|-----------|-----------|
| 안정(0) | 898 | 48.6% | +1.8% | 689 | 36.6% | +7.5% |
| 주의(1-2) | 853 | 46.4% | +2.7% | 891 | 40.6% | +4.2% |
| **고위험(3+)** | **18** | **50.0%** | **-19.0%** | **34** | **50.0%** | **+0.1%** |

고위험 등급은 두 연도 모두 base rate보다 하락 비율이 높고 평균 성장이 최저.
다만 n=18~34로 통계적 유의성에는 도달하지 못함.

### 개별 시그널: 우발부채(p=0.055), 계열변화(+18.3%p)가 가장 유망

- 우발부채↑: 2024에서 p=0.055 (5% 근접), 하락 42% vs 37%
- 계열변화: 이전 상승 기업 중 활성 시 하락 52.1% vs 33.8% (2024, n=48)
- 두 시그널 모두 2023에서는 재현 안 됨 → 1/2 일관성

## 106 실험 최종 결론

### 확인된 사실

1. **연간 재무 숫자로 매출 방향 예측은 불가** (001-004, 50±3%)
2. **문서 변화 크기(sizeDelta)로도 불가** (006, 모든 구간 동일)
3. **단순 키워드 카운팅으로도 불가** (007-008, 교차 검증 실패)
4. **disclosureRisk 고위험 등급이 유일하게 두 연도 일관** — 하지만 n 부족

### 이 실험이 dartlab에 남기는 것

- **disclosureRisk scan 엔진**은 방향이 맞다. 고위험 등급의 하락 편향이 관측됨
- 예측 도구로 쓰기엔 아직 이르지만, **위험 경고 도구**로서의 가치는 확인
- 향후 데이터 축적(2025, 2026)으로 n이 쌓이면 통계적 유의성 도달 가능
- **"안 된다"를 정직하게 확인한 것** 자체가 이 실험의 가치

### 다음에 재개할 때의 출발점

- disclosureRisk 고위험 등급: 연도 추가 시 재검증
- 우발부채/계열변화: p=0.055~0.066 — 데이터 1년 추가 시 유의 도달 가능
- LLM 기반 의미 분석: 완전히 다른 차원, 별도 실험으로 설계
