## 주요 발견 및 해결

### 헤더 패턴 다양성
1. **표준**: `품목 | 매입액 | 비율` — 대부분
2. **합쳐진 셀**: `매입액(비율)` — LG 등 → 괄호 분리 파싱
3. **분할 헤더**: row1 `품목 | 구분`, row2 `매입액 | 비율` — SK, F&F → 2행 병합
4. **연도 헤더**: `제N기 | 제N-1기` (매입액 키워드 없음) — 달바글로벌 → 연도 패턴 감지
5. **원ㆍ부재료**: 중간점(ㆍ) 변형 — 키워드 추가

### Shifted 행 문제
- segment 생략 시 데이터가 왼쪽으로 밀림 → `ratioVal > 100` 감지, `prevIdx` 숫자 확인
- 숫자만 있는 item → 순수 숫자 필터로 제거
- 값 없는 shifted 행 (대시만) → segment 업데이트 후 skip

### 기타 엣지 케이스
- `(주N)` 각주 참조 → parseAmount에서 None 반환
- 생산능력/수량 테이블 → isHeader에서 제외
- amount 컬럼 없는 헤더 → inTable 진입 차단
