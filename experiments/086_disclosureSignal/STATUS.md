# 086_disclosureSignal — 공시 텍스트 선행 신호 실험

## 목표
공시 텍스트 변화가 미래 실적을 예측하는지 실증

## Phase 3 GATE 판정: ✗ FAIL (텍스트 길이 프록시) / ✓ 부분 PASS (토픽 세분화)

- 001 관문: 사업의내용 전체 변화율 ρ=0.068 → FAIL (기준 >0.15)
- 002 발견: 토픽 세분화 시 14개에서 |ρ|>0.15. "경영진단"(ρ=0.284) 최고 예측력
- 003 발견: 범용 키워드(AI, ESG)는 역방향. 도메인 특화(바이오 +10.4pp)만 유효
- **결론**: 텍스트 "길이" 시계열은 부적합. 토픽 세분화 + 의미 기반 접근 필요

2. **지배구조 섹션 변화율** = 부정적 시그널 (주주의결권 ρ=-0.206, 대주주거래 ρ=-0.190)

4. **도메인 특화 키워드** + 기업 규모 정규화 필요
5. 텍스트 "양" 변화보다 **"의미" 변화** (semantic shift) 접근이 본질
