## Phase A 종합 결론 — 경쟁그룹 발견 ✅

**TF-IDF 기반 사업보고서 텍스트 유사도로 "진짜 경쟁그룹"을 발견할 수 있다.**

### 핵심 수치
| 지표 | TF-IDF peer | KIND 업종 | 랜덤 |
|------|-----------|----------|------|
| 매출 상관 | **0.27** (멀티토픽) | 0.16 | 0.08 |
| 강한 상관(>0.3) | **46.3%** | 34.8% | 15.5% |

### 방법론

2. **멀티토픽 결합** (biz 0.5 + seg 0.25 + cont 0.15 + risk 0.10) → 단독 대비 +0.04
3. **시간 안정성** 확인: Jaccard 0.68, 상위 3 peer 75% 유지
4. **매출 커버리지 한계**: 177/2408개 (account_nm 매칭 확대 필요)

## Phase B 종합 결론 — 거시 시나리오 ⚠️

**GDP 탄성도의 방향성은 유효하나, 절대 크기와 예측력에 근본적 한계.**

### 핵심 수치
| 지표 | 결과 |
|------|------|
| 섹터 탄성도 방향 일치 | **88%** (7/8) |
| 실증β vs 설정β 상관 | 0.169 (약함) |
| 하드코딩 beta MAE | **11.4%** |
| 실증 beta MAE | 23.4% (악화) |
| regex 수출비중 추출률 | 29% (실용 한계) |

### 발견
1. **하드코딩 beta의 보수적 크기(~1.0)가 오히려 안정적 예측** — 실증 beta는 과적합
2. **GDP 단일변수로는 매출 변동의 30%만 설명** (R² 평균 0.31)
3. **구조적 충격(코로나)은 GDP로 포착 불가** — 산업별 비대칭 효과
4. **보복소비/수요이전 같은 2차 효과**가 1차 거시충격보다 클 수 있음
5. **regex 기반 수출비중 추출은 29%** — 테이블 파싱 또는 LLM 추출 필요

### 시사점
- 거시 시나리오는 방향성 참고용으로 유효, 정밀 예측에는 부족
- 기업별 감응도는 sections에서 테이블 기반 추출(수출비중, 원재료 비중)이 필요
- simulation.py beta는 현 수준 유지하되, 실증 교정보다는 regime 구분이 우선

## Phase C 종합 결론 — Peer 기반 예측 ✅

**TF-IDF peer consensus는 가장 강력하고 안정적인 매출 예측 앵커.**

### 핵심 수치
| 지표 | Peer consensus | Naive | Sector median | Regression |
|------|--------------|-------|---------------|------------|
| MdAE | **9.0%** | 13.7% | 12.3% | 11.1% |
| 20% 이내 | **84.7%** | 64.9% | 71.8% | 69.3% |
| peer 내 R² | — | — | — | **0.319** |

### 발견
1. **Peer consensus(중앙값)가 최고 성능** — MdAE 9.0%, 20%이내 84.7%
2. **Regression R²=0.319** — peer 한정 시 회귀 설명력 부활 (전체 R²=0.00 대비)
   단 out-of-sample은 consensus에 열등 (5시점 OLS 과적합)
3. **Peer 쌍의 72%가 유의미 상호작용** — 양의 상관(49%) > 음의 상관(23%)
4. **동행이 제로섬보다 지배적** — peer consensus가 유효한 근본 이유
5. **성숙시장(통신)에서만 약한 제로섬** — KT-SKT r=-0.307

### 시사점
- peer consensus를 예측 앙상블의 핵심 소스로 편입 가능
- 중앙값 기반이 가중/회귀보다 robust (이상치 방어)
- 매출 커버리지 확대(132→2000+)가 실용화의 최대 과제

## Phase D 종합 결론 — 최종 벤치마크 ✅

**avg3y + peer consensus 앙상블이 최적. MdAE 12.0% (3개년 평균).**

### 최종 성능표 (3개년 평균)
| 모델 | MdAE | <20% | 방향 적중 |
|------|------|------|---------|
| **ensemble2 (avg3y+peer)** | **12.0%** | **69.5%** | **66.3%** |
| ensemble3 (+sector) | 12.1% | 68.5% | 63.4% |
| avg3y | 13.2% | 65.2% | 61.3% |
| peerConsensus | 13.3% | 67.3% | 64.2% |
| naive | 14.5% | 61.3% | 63.1% |
| sectorMedian | 15.6% | 61.8% | 59.4% |

### 발견
1. **2개 소스(avg3y+peer)가 최적** — sector 추가 효과 미미
2. **peer가 최대 한계 기여** (제거 시 +1.2%p 악화)
3. **macro(GDP beta)는 기여 없음** — 제외 시 오히려 개선
4. 3개년 모두 앙상블이 모든 단일 기준선을 능가

### 외부 벤치마크 위치
- 애널리스트 consensus: MdAE ~5-8% (정보 우위)
- **dartlab 앙상블: MdAE 12.0%** (순수 통계)
- ML(LSTM/RF): MAE ~10-15% (대규모 학습 데이터 필요)
- Naive/Sector: MdAE ~15% (우리 결과와 일치)

## Phase E 종합 결론 — ECOS/FRED 다변량 거시 ❌

**거시 변수는 연간 8개 관측치 수준에서 매출 예측에 기여 불가.**

### 핵심 수치
| 실험 | 방법 | 결과 |
|------|------|------|
| 015 | 5변수 다변량 회귀 | R²=0.788(in) → MdAE 20.7%(out). GDP단독 11.0%보다 2배 나쁨 |
| 016 | GDP+FX 2변수 앙상블 | MdAE 12.1% (기존 12.0%). 커버리지 7%. 기여 0 |
| 017 | 선행지표 시차 상관 | 방향 48.2% = 동전 던지기. lag r 높아도 예측력 없음 |
| 018 | 시나리오 프로토타입 | 방향 67%, MAE 8.1%p. 정상경기만 유효(88%) |

### 발견
1. **n=8 연간 데이터의 근본 한계**: 5변수 회귀 과적합, 2변수도 효과 없음, 상관도 spurious
2. **선행 상관 ≠ 예측력**: |r|>0.5인 lag 조합도 방향 예측 0% (음의 상관 함정)
3. **하드코딩 탄성도 > 실증 회귀**: 보수적 beta가 과적합 방지에 유리 (MAE 8.1 vs 20.7)
4. **시나리오 도구는 방향성 참고용**: 정상 경기 88%, 구조적 충격 50%
5. **거시가 유효하려면**: 월별 데이터(n=100+) 또는 패널 회귀 필요

### 시사점
- simulation.py의 하드코딩 SECTOR_ELASTICITY는 현 수준 유지 (실증 교정보다 안정)
- 시나리오 프리셋은 방향성 UI 도구로 가치 있음 (정밀 예측 아닌 what-if)
- **앙상블에 거시 추가는 하지 않는다** — Phase B+E 양쪽에서 확인
