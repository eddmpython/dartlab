# 017-020 Source Checklist

다음 배치 글의 공식 출처 확인 체크리스트다.

## 017. 20-F와 10-K는 무엇이 다른가

- SEC Forms Index
- SEC investor / filing explainer
- 20-F, 10-K, 6-K 공식 form 설명

## 018. Risk Factors와 MD&A를 같이 읽는 법

- SEC `How to Read a 10-K`
- 10-K form 구조 설명
- 사업보고서 / MD&A 관련 감독기관 안내

## 019. 재고자산과 평가손실 읽는 법

- 사업보고서 주석 구조
- 재고자산 관련 회계기준 설명 자료
- 감사보고서 KAM 예시

## 020. XBRL 재무제표 원문과 주석 다운로드 파이프라인

- OpenDART 개발가이드
- XBRL 재무제표 관련 가이드
- 주석 다운로드 / 원문 다운로드 관련 공식 문서
