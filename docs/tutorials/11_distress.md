# 11. 부실 예측 스코어카드

기업의 부실 위험을 4축 가중 평균으로 종합 판정하고, 신용등급(AAA~D)과 유동성 경보를 포함한 근거 기반 레포트를 생성한다.

## 빠른 시작

```python
import dartlab

c = dartlab.Company("005930")
print(c.insights.distress)
```

출력 예시:

```
=== 부실 예측 스코어카드 ===
종합: safe (2.1/100) | 신용등급: AAA (투자적격 최상위)

[정량 분석] 1.3/100 (가중 40%)
  Ohlson O-Score       P(부도) 0.0%     safe       부도 확률 극히 낮음. 재무구조 건전.
  Altman Z''-Score     Z'' = 8.42       safe       비제조업 기준 안전 영역.
  Altman Z-Score       Z = 5.12         safe       제조업 기준 안전 영역.

[이익 품질] 0.0/100 (가중 20%)
  Beneish M-Score      M = -3.45        safe       이익 조작 가능성 낮음. 회계 품질 양호.

  시계열 악화 패턴 없음.
```

## 방법론

### 4축 구성

| 축 | 가중치 | 구성 모델 | 출처 |
|----|--------|-----------|------|
| 정량 분석 | 40% | O-Score, Z''-Score, Z-Score | ratios.py 계산 |
| 이익 품질 | 20% | Beneish M-Score, Sloan Accrual, Piotroski F-Score | ratios.py 계산 |
| 추세 분석 | 30% | 연속적자, ICR 1 미만, 부채비율 상승, CCC 확대 | anomaly 탐지기 |
| 감사 위험 | 10% | 비적정 감사의견, 지배구조 이상 | anomaly 탐지기 |

각 축은 0~100 정규화 (0=안전, 100=위험). 종합 점수 = 가중 평균.

### 레벨 판정

| 종합 점수 | 레벨 | 의미 |
|-----------|------|------|
| 15 미만 | safe | 안전 |
| 15~29 | watch | 관찰 |
| 30~49 | warning | 경고 |
| 50~69 | danger | 위험 |
| ≥ 70 | critical | 심각 |

## 정량 모델 상세

### Ohlson O-Score (1980)

9변수 로지스틱 회귀 모델. 학술 적중률 96.1%.

**수식:**
```
O = -1.32 - 0.407×log(TA/GNP)
    + 6.03×(TL/TA)
    - 1.43×(WC/TA)
    + 0.076×(CL/CA)
    - 1.72×(TL>TA ? 1 : 0)
    - 2.37×(NI/TA)
    - 1.83×(OCF/TL)
    + 0.285×(INTWO)
    - 0.521×(CHIN)

P(부도) = 1 / (1 + exp(-O))
```

- `INTWO`: 최근 2년 연속 순손실이면 1
- `CHIN`: NI 변화율 `(NI_t - NI_{t-1}) / (|NI_t| + |NI_{t-1}|)`

### Altman Z''-Score (1995)

비제조업/신흥시장 변형 4변수 모델.

```
Z'' = 6.56×(WC/TA) + 3.26×(RE/TA) + 6.72×(EBIT/TA) + 1.05×(BV_Equity/TL)
```

| Z'' | 영역 |
|-----|------|
| > 5.0 | 안전 |
| 2.6~5.0 | 회색 |
| 1.1~2.6 | 부실 위험 |
| 1.1 미만 | 부실 |

### Beneish M-Score (1999)

8변수 이익 조작 탐지 모델.

| M-Score | 판정 |
|---------|------|
| > -1.78 | 조작 가능성 높음 |
| -2.22 ~ -1.78 | 회색 영역 |
| -2.22 미만 | 안전 |

### Piotroski F-Score (2000)

9항목 바이너리 펀더멘탈 강도 점수.

| F-Score | 판정 |
|---------|------|
| 7~9 | 강건 |
| 5~6 | 보통 |
| 3~4 | 취약 |
| 0~2 | 심각 |

### Sloan Accrual Ratio (1996)

발생주의 이익 비중. |비율| > 10% 시 이익 품질 의심.

## 신용등급 매핑

종합 점수를 S&P 부도 확률(PD) 대응표 기반으로 신용등급에 매핑.

| 점수 범위 | 등급 | 설명 |
|-----------|------|------|
| 0~4 | AAA | 투자적격 최상위 |
| 5~9 | AA | 투자적격 상위 |
| 10~14 | A | 투자적격 |
| 15~24 | BBB | 투자적격 하한 |
| 25~34 | BB | 투기등급 |
| 35~49 | B | 투기등급 하위 |
| 50~64 | CCC | 상당한 부실 위험 |
| 65~79 | CC | 부실 임박 |
| 80~89 | C | 부도 직전 |
| 90~100 | D | 부도 수준 |

---

## 유동성 경보

현금 소진 예상 개월 수를 계산한다.

- **영업CF 양수**: 소진 없음 (현금 축적 중)
- **영업CF 음수**: `현금 / (|월간 영업CF|)` → 소진 개월 수
- **영업CF 없음**: `현금 / (월간 매출원가 + 판관비)` → 소진 개월 수

| 개월 수 | 경보 |
|---------|------|
| 24+ | 충분 |
| 12~24 | 양호 |
| 6~12 | 주의 |
| 3~6 | 경고 |
| 3 미만 | 위험 |

---

## 업종별 주의사항

- **금융업**: Z-Score 적용 불가 (매출/유동자산 계정 구조 상이). O-Score, Z''-Score만 사용.
- **지주회사**: DSO 극단값 빈출 — CCC 탐지기 자동 제외.
- **바이오**: DIO 극단값 — CCC 탐지기 주의.
- **반도체**: CCC 구조적 장기 — 업종 특성으로 해석.

---

## 데이터 품질

| 모델 수 | 판정 |
|---------|------|
| 5개+ | 충분 |
| 3~4개 | 보통 |
| 1~2개 | 부족 |

데이터 부족 시 계산 가능한 모델만으로 축 점수를 산출한다. 모델이 0개인 축은 0점으로 처리.

## 프로그래밍 접근

```python
d = c.insights.distress

# 종합 판정
print(f"Level: {d.level}, Grade: {d.creditGrade}")

# 개별 축 접근
for axis in d.axes:
    print(f"\n{axis.name}: {axis.score}/100 (가중 {axis.weight*100:.0f}%)")
    for model in axis.models:
        print(f"  {model.name}: {model.displayValue} [{model.zone}]")
        print(f"    → {model.interpretation}")
        print(f"    참조: {model.reference}")

# 유동성
if d.liquidityAlert:
    print(f"\n유동성: {d.liquidityAlert}")

# 위험 요인
for rf in d.riskFactors:
    print(f"  ⚠ {rf}")
```
