## 분석 해석

### 핵심 스토리

LG화학은 **배터리(에너지솔루션) 적자가 화학 본업을 잠식**하는 기업이다. EBITDA 음수로 EV/Sales fallback이 적용되어 시나리오 간 목표가 차이가 작다. 화학(석유화학/첨단소재)과 배터리의 수익성이 정반대이지만 연결 기준으로 한 덩어리 처리.

### 수치 해석

- **EV/Sales fallback**: EBITDA 음수에서 매출 기반 terminal value 적용. 매출 차이가 크지 않으면 시나리오가 수렴
- **적자기업 DCF의 한계**: 현재 적자 → 정규화 수익(normalized earnings) 기반 DCF가 표준이나 미구현
- **사업부 분리가 핵심**: LG에너지솔루션은 이미 상장됨. 화학 본업만 별도 밸류에이션하면 더 정확

### 엔진 개선점

1. **보완**: 적자기업 normalized earnings DCF 구현 (과거 정상기 마진 × 현재 매출)
2. **보완**: 상장 자회사(LG에너지솔루션) 분리 SOTP 필요

## 분석 한계

- EBITDA 음수 기업으로 EV/Sales fallback 적용. 매출 기반 terminal value는 마진 구조를 반영하지 못해 시나리오 간 목표가 차이가 작음
- 배터리 사업부(에너지솔루션)와 화학/첨단소재 사업부의 성장성/수익성이 극명히 다르나 연결 기준 단일 DCF로 처리
- 배터리 업종 표준인 EV/매출 또는 용량(GWh) 기반 밸류에이션 미구현
- 2차전지 소재(양극재/분리막) 시장 성장률과 가격 경쟁 압력이 마진에 미치는 영향 미반영
